Hilfreiche Einstellungen

ℹ️

Für diese Einstellungen benötigst du folgende Erweiterungen:

Installiere sie in VS Code über Ctrl + Shift + X → Suche nach dem Namen.

  • Atom One Dark Theme
  • Material Icon Theme

So fügst du die Einstellungen in VS Code ein:

  1. Öffne die Command Palette mit Ctrl + Shift + P
  2. Gib "user" ein und wähle "Open User Settings (JSON)"
  3. Füge den kopierten Code in die settings.json ein
  4. Speichere die Datei (Ctrl + S)

        {
          // Arbeitsoberfläche (Workbench)
          "workbench.colorTheme": "Atom One Dark",         // Theme für Editor und UI (Farbschema)
          "workbench.iconTheme": "material-icon-theme",    // Visuelle Icons für Dateitypen im Explorer
          "workbench.startupEditor": "none",               // Beim Start keinen Editor öffnen (leeres Fenster)
          "workbench.tree.indent": 20,                     // Einrückung in Baumansichten (z.B. Explorer)

          // Editor
          "editor.fontSize": 20,                           // Schriftgröße im Editor
          "editor.minimap.enabled": false,                 // Miniaturansicht (Minimap) rechts deaktivieren
          "editor.stickyScroll.enabled": false,            // "Klebende" Code-Überschriften deaktivieren
          "editor.guides.bracketPairs": "active",          // Fügt Visuelle Linien bei verschachtelten Klammern hinzu
          "editor.guides.bracketPairsHorizontal": "active",// Horizontale Linien bei Klammern (zusätzlich zu vertikalen)
          "editor.tabSize": 2,                             // Tabulatorgröße (Leerzeichen pro Tab)

          // Terminal
          "terminal.integrated.fontSize": 20,              // Schriftgröße im integrierten Terminal

          // Datei-Management
          "files.insertFinalNewline": true,                // Fügt am Dateiende automatisch eine Leerzeile ein
          "files.trimFinalNewlines": true,                 // Entfernt überflüssige Leerzeilen am Dateiende
          "files.trimTrailingWhitespace": true,            // Automatisch Leerzeichen am Zeilenende entfernen

          // Updates & Benachrichtigungen
          "update.showReleaseNotes": false                 // Keine Release-Hinweise nach Updates anzeigen
        }
